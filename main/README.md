## 进入服务器
在启动器内下载**1.18**以上的任意版本或使用Creeper Client，启动游戏。

服务器的IP是 **craftspace.top** ，在多人游戏里输入即可。
::: warning
我们只对使用**1.18**及以上的玩家支持，虽然使用1.8及以上版本仍可进入服务器，但我们不推荐，这样会产生许多奇奇怪怪的Bug，并且0<Y<256的区域也会不会加载，这是无解的。
:::

### 注册
进入服务器即可，无需注册。
::: details 如果你是离线玩家
进入服务器后按照提示输入`/register <密码>`进行注册即可。
:::
